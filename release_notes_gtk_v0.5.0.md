# Floss Finder GTK v0.5.0

Experimental native Linux GTK build of Floss Finder.

## Changes

- Added native GTK Linux interface
- Added FlossKeeper stash loading
- Added stash file picker
- Added clear stash button
- Added Settings button
- Added dark mode toggle
- Improved dark mode styling
- Added softer dark-mode stash text box
- Added GTK Linux installer script

## Notes

This is a Linux GTK test release from the `native-linux-ui` branch.

The stable egui version remains on `master`.
